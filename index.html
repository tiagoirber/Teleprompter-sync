Funcionalidades

Tudo que você precisa
para gravar sozinho

Construído especificamente para criadores que gravam sem equipe, com foco em estabilidade e UX durante a gravação real.

Rolagem por voz

Sincronização em tempo real usando Web Speech API com algoritmo de matching por bigramas — tolerante a pausas, improvisos e variações de dicção.

Tela cheia real

Modo fullscreen com controles que somem automaticamente após 3s, linha de leitura configurável e gradientes de fade nas bordas.

Contagem regressiva

3s ou 5s antes de iniciar a rolagem. Dá tempo para posicionar a câmera e se preparar sem precisar de um assistente.

Modo foco

Apenas 3, 5 ou 7 linhas visíveis ao redor da linha ativa. O resto some progressivamente para eliminar distração visual.

Alto contraste

Tema amarelo sobre preto para uso com vidro de teleprompter profissional ou em ambientes com iluminação intensa.

Espelhamento horizontal

Inverte o texto horizontalmente para uso com sistemas de teleprompter profissional com espelhos semitransparentes.

Editor completo

Múltiplos roteiros, renomear, duplicar, excluir com confirmação, contagem de palavras, tempo estimado de leitura e auto-save.

100% local

Tudo salvo no localStorage. Nenhum dado sai do seu browser. Funciona offline para rolagem manual — voz requer conexão.

Como funciona

O algoritmo de sincronização

Quatro camadas independentes transformam áudio bruto em posicionamento preciso no roteiro.

01

Normalização

Remove acentos, pontuação e palavras de preenchimento ("então", "tipo", "né") do transcript e do roteiro antes de comparar.

02

Similaridade por bigramas (Dice)

Compara pares consecutivos de palavras em vez de caracteres. Robusto a palavras omitidas, trocadas ou inseridas. Bigramas dos chunks pré-computados — zero alocação durante a gravação.

03

Janela móvel com bias de avanço

Busca apenas nas ±N linhas ao redor da posição atual. Linhas à frente recebem +15% no score — modela que o usuário quase sempre avança. Fallback global só quando necessário.

04

Filtro de confirmação

Um novo índice só é aceito após aparecer como melhor match por 2 resultados consecutivos. Scores > 0.72 passam direto. Recuos > 3 linhas são bloqueados.

Instalação

Pronto em 5 minutos

Clone, instale e rode. Sem configuração de backend, sem variáveis de ambiente obrigatórias.

1

Clone e instale dependências

git clone https://github.com/seu-usuario/teleprompter
cd teleprompter
npm install
2

Adicione os componentes shadcn/ui

npx shadcn@latest init
npx shadcn@latest add button slider label switch \
  textarea tooltip separator dropdown-menu alert-dialog
3

Rode em desenvolvimento

npm run dev
# Abra http://localhost:3000
4

Deploy no Netlify

Conecte o repositório no painel do Netlify. Build command: npm run build · Publish directory: .next
